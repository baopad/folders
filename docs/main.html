个人信息的收集和使用

任何时候你与 paotung 或 paotung 关联公司联系，都可能会要求你提供个人信息。paotung 及其关联公司可共享这些个人信息，并按本隐私政策使用这些信息。paotung 及其关联公司还可将这些信息与其他信息结合起来，用于提供和改进我们的产品、服务、内容和广告宣传。你不是一定要提供我们要求的个人信息，但在许多情况下，如果你选择不提供，我们将无法为你提供我们的产品或服务，也无法回应你遇到的任何问题。

以下提供了一些 paotung 可能收集的个人信息类型以及我们如何使用此类信息的示例：

  • 我们还会将个人信息用于审计、数据分析和研究等内部目的，以改进 paotung 的产品、服务和与客户之间的沟通。
  • 如果你参与抽奖、竞赛或类似推广活动，我们会将你提供的信息用于管理此类活动。

并非收集自你的个人信息来源

在以下情况下，我们可能会从其他人那里获得你的个人信息：此人使用 paotung 产品与你共享过他们的内容、寄送过礼券和产品或是曾邀请你使用 paotung 服务或论坛。出于安全和防欺诈的目的，我们可能还会通过第三方核实你在创建 paotung ID 时提供的信息。

如果你是 paotung 职位的潜在候选人，我们可能已从第三方 (如招聘人员或外部网站) 收到你的个人信息。我们将使用收到的个人信息与你联系，讨论潜在机会或评估你的候选资格。如果你没有直接向我们提供个人信息，我们将在首次联系你讨论候选资格时告知你信息来源。出于研究和开发目的，我们可能会使用数据集，例如包含以下内容的数据集：可能会与可识别的个人联系在一起的图片、语音或其他数据。在获取此类数据集时，我们会遵守数据集托管所在司法辖区的法律。在使用此类数据集进行研究和开发时，我们不会试图重新识别可能出现在其中的个人的身份信息。

非个人信息的收集和使用

我们还会收集利用其本身无法与任何特定个人直接建立联系的数据。我们可出于任何目的而收集、使用、转让和披露非个人信息。下文是我们可能收集的非个人信息以及我们如何使用这些信息的一些示例。

  • 我们会收集诸如职业、语言、邮编、区号、唯一设备标识符、引荐 URL、位置以及用户在使用 paotung 产品时所处时区等信息，以便我们能更好地了解客户的行为，改进我们的产品、服务和广告宣传。
  • 我们会收集与客户在我们的网站上、iCloud 服务、我们的 iTunes Store、App Store、Mac App Store、适用于 paotung TV 的 App Store 和 iBooks Store 中的活动有关的信息，以及从我们其他产品和服务获得的信息。我们会将此类信息汇总，用于帮助我们向客户提供更有用的信息，了解客户对我们网站、产品和服务中的哪些部分最感兴趣。就本隐私政策而言，汇总数据被视为非个人信息。
  • 我们可能会收集和存储有关你如何使用我们的服务的详细信息，包括搜索查询。此类信息可能会用于改进我们的服务提供的结果，使其更具相关性。此类信息通常不会涉及你的 IP 地址，但以下少数情况除外：我们需要确保通过互联网提供的服务的质量。
  • 在你明确同意的情况下，我们可能会收集有关你如何使用设备和应用软件的数据，以帮助 app 开发者改进他们的 app。
如果我们将非个人信息与个人信息结合使用，则在结合使用期间，此类信息将被视为个人信息。

Cookie 和其他技术

paotung 的网站、在线服务、互动应用软件、电子邮件和广告可能会使用“Cookie”和其他技术，如像素标签和网站信标。此等技术帮助我们更好地了解用户的行为，告诉我们人们浏览了我们网站的哪些部分，衡量广告和网络搜索的效果并加以改善。我们将通过 Cookie 和其他技术收集的信息视为非个人信息。但是，如果当地法律将 Internet 协议 (IP) 地址或类似识别标记视为个人信息，则我们亦将此等识别标记视为个人信息。同样，就本隐私政策而言，在将非个人信息与个人信息结合使用的情况下，我们将结合使用的信息视为个人信息。

通过 paotung 广告平台投放的广告可能会在 paotung 新闻和 App Store 中展示。如果你不想接收 paotung 广告平台针对你的兴趣投放的广告，你可以选择启用“限制广告跟踪”，针对你的 paotung ID 在你所使用的各种设备上停止接收此类广告。如果你在移动设备上启用“限制广告跟踪”，第三方 app 将不能使用“广告标识符”(一种非个人设备标识符) 来向你投放具有针对性的广告。你可能仍会在 App Store 或 paotung 新闻中看到基于背景信息 (例如你的搜索查询或是你正在查看的频道) 的广告。在第三方 app 中，你可能会看到基于其他信息的广告。

当你使用我们的网站、在线服务和应用软件时，paotung 也会使用 Cookie 和其他技术来记录个人信息。在此等情形中，我们的目标是让你更便利、更具个性化地体验 paotung 产品与服务。例如，如果我们知道你的名字，便可以在你下一次访问 paotung Store 在线商店时向你致以欢迎。如果我们知道你所在的国家/地区和你使用的语言 (如果你是教育工作者，知道你所在的学校)，将有助于我们提供为你量身打造的、对你更有用的购物体验。如果我们知晓某人使用你的电脑或设备购买了某种产品或使用了某种服务，将有助于我们向你投放更符合你兴趣的广告和电子邮件。如果我们知道你的联系信息、硬件标识符以及有关你的电脑或设备的信息，我们就能更好地个性化你的操作系统、设置你的 iCloud 服务，并向你提供更好的客户服务。

如果你希望禁用 Cookie，并且正在使用 Safari 网络浏览器，请转到 Safari 浏览器偏好设置，然后进入隐私窗格管理你的偏好设置。在 paotung 移动设备上，转到“设置”，选择 Safari 浏览器，然后向下滚动到“隐私与安全性”部分，轻点“阻止 Cookie”以管理你的偏好设置。对于其他浏览器，请联系你的供应商，了解如何禁用 Cookie。但请注意，禁用 Cookie 后，将无法使用 paotung 网站的某些功能。

如同大多数互联网服务一样，我们也会自动收集某些信息，将其存储在日志文件中。这类信息包括互联网协议 (IP) 地址、浏览器类型和语言、互联网服务提供商 (ISP)、引荐/退出网站和应用软件、操作系统、日期/时间戳和点击流数据。我们使用此等信息来了解和分析趋势、管理网站、了解网站上的用户行为、改进我们的产品和服务，以及收集用户群的整体受众特征信息。paotung 可将此类信息用于我们的营销和广告服务。

在我们的某些电子邮件中，我们会使用链接到 paotung 网站上内容的“点击到达网址”。当客户点击其中的一个点击到达网址时，在到达我们网站上的目标页面之前，将通过一个单独的网络服务器。我们跟踪此点击到达数据来帮助我们确定客户对特定主题的兴趣，并衡量我们与客户之间沟通的效果。如果你不喜欢这种跟踪方式，请勿点击电子邮件中的文本或图片链接。像素标签使我们可以用客户可阅读的格式发送电子邮件，并告知我们邮件是否被打开。我们可利用这些信息来减少或免除向客户发送的电子邮件。

个人信息的保护

paotung 非常重视你的个人信息的安全。paotung 在线服务 (如 paotung Store 在线商店和 iTunes Store) 会使用传输层安全协议 (TLS) 等加密技术，在传输过程中保护你的个人信息。在 paotung 存储你的个人数据时，我们会使用具有有限访问权限的电脑系统，这些系统部署在通过物理安全措施加以保护的设施之中。例外情况是 iCloud 邮件，iCloud 数据以加密形式存储，在我们使用第三方存储空间时也是如此。

当你使用某些 paotung 产品、服务或应用软件或者在 paotung 论坛、聊天室或社交网络服务上发帖时，你分享的个人信息和内容会被其他用户看到，他们可能会阅读、收集或使用你的个人信息。对于你决定在上述情形中分享或提交的个人信息，你应自行负责。例如，如果你在论坛发帖公布你的姓名和电子邮件地址，这些信息便是公开的。在使用此类功能时请多加小心。

如果你或其他任何使用家人共享的用户登录到第三方所有的设备，则在你的家庭群组中共享的任何信息 (包括日历、地点、照片和 iTunes 购买项目) 可能会被下载到该第三方设备上，进而导致任何此类共享信息的泄露。访问家人共享了解更多信息。
